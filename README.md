# curso-go-gRPC
Repositório do curso gRPC em Go.

## Instalação do PROTOBUF

- [x] sudo apt-get install -y git autoconf automake libtool curl make g++ unzip
- [x] git clone https://github.com/google/protobuf.git
- [x] cd protobuf/
- [x] ./autogen.sh
- [x] ./configure
- [x] make
- [x] make check
- [x] sudo make install
- [x] sudo ldconfig
- [x] protoc -h
